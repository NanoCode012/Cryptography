# Cryptography

## Requirements

```
python >= 3.8
```

## Setup

```bash
pip install -r requirements.txt
python setup.py build_ext --inplace
```

## Usage

```bash
python app.py [optional parameters]
```

See `app.py` ArgumentParser for optional parameters.

## Test
Easily test that all features are working via
```bash
python tests.py [-v] 
```

`-v`: verbose
